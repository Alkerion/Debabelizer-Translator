About this extension

Debabelizer Translator is a lightweight Firefox extension that brings Google Translate directly into your browser toolbar. Type any text, select it on a page, or use the right-click context menu to translate instantly ‚Äî without opening a new tab.

This is a maintained fork of the original Google‚Ñ¢ Translator extension by Andy Portmen, updated for compatibility with current versions of Firefox. The Google Translate API endpoint has been corrected, deprecated code removed, and the manifest updated to meet current Mozilla standards.

Note: This extension is not affiliated with or endorsed by Google LLC.

Features

  • One-click translation via the toolbar popup
  • Translate selected text by holding Alt or Insert key
  • Pop-up bubble translator on any webpage
  • Right-click context menu: translate selected text or full page
  • Auto-detection of source language
  • Translation history with quick recall
  • Text-to-speech for translated content
  • Swap source and target languages in one click
  • Supports all languages available in Google Translate
